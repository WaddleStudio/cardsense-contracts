# CardSense Contracts

Shared data contracts for the CardSense platform.

This repository defines the **JSON schemas and taxonomies**
used by both the CardSense API and the CardSense Extractor.

## What this repo does
- Defines normalized promotion data structures
- Defines category and channel taxonomies
- Defines versioning and breaking-change rules

## What this repo does NOT do
- No crawling or data extraction
- No API implementation
- No database code

## Related Repositories
- cardsense-api — External decision API
- cardsense-extractor — Promotion extraction pipeline
